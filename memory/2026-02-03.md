**Harvard Study on Selective Recall** (d3.harvard.edu)
- "Experience-Following" pattern: when agent task resembles stored memory, outputs match closely
- This accelerates learning when memory is correct, BUT if bad memories are stored, they propagate and degrade performance
- Key insight: memory QUALITY matters more than quantity - bad memories don't just linger, they "creep" into future decisions
- Strategic addition/deletion of memories is crucial for long-term agent performance
- Source: "How Memory Management Impacts LLM Agents: An Empirical Study of Experience-Following Behavior" by Lakkaraju et al.
